SVG vs Icon Fonts

SVG vs Icon Fonts

  • Common problems with font support;
  • Different browser — different render;
  • Usage of ::before and ::after pseudo elements;
  • Difficult stylization — only color, shadow, size;
  • No multicolor ability;
  • Problems in case on fail font loading, during the font and css loading.

SVG vs Icon Fonts

Seren Davies: Death to icon fonts - EpicFEL 2015

SVG vs JPG/PNG

  • JPG/PNG are impossible to stylize;
  • For modern screens you need to make @2x, @3x sizes;
  • Often you have to choose between good view and big size.

How to use SVG?

Image

				<img src="apple.svg" alt="Apple" />
			</div>

How to use SVG?

Inline svg

			<div>
				<svg ... >
					<line .... />
				</svg>
			</div>

How to use SVG?

Embed, object, iframe

			<div>
				<iframe src="apple.svg" ... > </iframe>
			</div>

How to use SVG?

SVG sprites

			<svg>
				<defs>
					<symbol id="pinapple"><line .../></symbol>
					<symbol id="grape"><line .../></symbol>
					<symbol id="orange"><line .../></symbol>
				</defs>
			</svg>

How to use SVG?

SVG sprites

			<div>
				<svg>
					<use xlink:href="fruits.svg#pinapple" />
				</svg>
			</div>

How to use SVG?

In CSS as background-image
